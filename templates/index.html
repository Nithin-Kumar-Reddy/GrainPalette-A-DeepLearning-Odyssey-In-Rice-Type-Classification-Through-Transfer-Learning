About the Rice Type Classification Model

This model is built using a Convolutional Neural Network trained on labeled images of rice grains. It predicts the rice type based on the image.

  • Accuracy: 97%
  • Types Detected: 5 rice varieties
  • Dataset: Kaggle dataset with labeled images
  • Architecture: MobileNetV4 with Transfer Learning
